Visual Design and User Experience

When addressing Visual design and the look of websites, users typically form their first impression within the first 50 milliseconds. We typically associate poorly designed websites with malicious activities and therefore, trust those websites less than a professionally designed website. This will then determine if they want to stay and explore further or leave the site. For this reason the design and feel of a website is important to keep the user's attention and attract more traffic as well as to encourage to explore. In this module, Visual Design and user experience, students will learn how to design websites using HTML and CSS.

Module Aims

  • To communicate knowledge and understanding of the basic concepts of graphical user interface (GUI) design
  • To identify design guidelines and demonstrate applications that use them
  • To demonstrate a number of different visual interface environments (desktop GUI, web, mobile phone interfaces)
  • To demonstrate how to prototype a user interface

Module Content

  • Principles of good user interface design
  • Concepts of visual interfaces
  • Applying the guidelines
  • The design process
  • Building a prototype
  • Evaluating a design
  • Creating, editing and using images for websites and other intefaces
